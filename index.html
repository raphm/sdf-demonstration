Use the 'a' and 'd' keys to move the text left and right.

Use the 'w' and 's' keys to move the text towards and away from you.

Use the 'q' and 'e' keys to move the text up and down.

Use the 'g' key to turn on glowing borders. Use the 'h' key to turn off the glowing borders.

The cube code in this example is heavily altered, but had its origins in code from Lesson 8 from LearningWebGL.com.
